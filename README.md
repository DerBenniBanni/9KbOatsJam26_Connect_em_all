# 9KbOatsJam26_Connect_em_all
Jam entry for Where is my oats 9kb gamejam 2026

**Theme:** "Connections"

**Game size (zipped, total):** 3,58 KB (no further compression like roadroller used, only zipped)

**How to play:** 

Connect the numbered dots with lines, and don't cross your lines!

Use the "Shuffle" button in the top right corner to retry

**AI usage:** The game was 100% created using opencode (llm: big-pickle) in about an hour, as i wanted to try this approach. I am surprised by the outcome... (and frightened aswell)

**SFX:** created using pro.sfxr.me
